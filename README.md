# TSA 2025 Robocode
Welcome to the run adventures that are Robocode!!

## Viktor's Description from ChatGPT
Robocode is a programming game where players write AI-controlled tanks in Java to battle in an arena. 

The goal is to develop the best possible strategy for your tank, which involves movement, targeting, dodging, and firing at opponents. 

The game provides a simulated environment where tanks use sensors to detect enemies and make decisions based on programmed logic.

## How do I get started?
1. Install the Robocode environment following the instructions.
2. Take a look at the TSARobot.java file and the documentation.
